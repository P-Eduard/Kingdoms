Kingdoms general info.

  • This is a simulation to see which kingdom will prevail.
  • The initial grid can be setup by clicking the squares and circling between the available kingdoms.
  • In the beginning, no opposing kingdoms can be neighboring.
    This is to give each kingdom a chance to set foot on the map.
  • Each kingdom is looked at as the individual set of lands that compose it.
  • Each set of lands will pick a square to expand and send power to the square equal to the area they own.
  • If the square picked is defending the defender will have +1 power as it's easier to defend than to attack.
  • Resizing the grid will reset the game.
